VELARION LUMEN — CONFIGURAÇÃO RÁPIDA

Arquivo principal de configuração:
assets/js/velarion.config.js

1) BACKGROUNDS DO LOADER
Coloque as imagens em:
assets/img/loading/

Use nomes sequenciais:
background1.png
background2.png
background3.png
...

Depois edite apenas:
loading.backgroundCount

Exemplo:
Se tiver background1.png até background40.png:
backgroundCount: 40

2) MANUTENÇÃO
No mesmo arquivo:
maintenance: {
  enabled: false,
  page: "maintenance.html"
}

Troque enabled para true:
enabled: true

Quando ativo, as páginas redirecionam para maintenance.html.

Importante:
Como este projeto é HTML/CSS/JS estático, esse bloqueio é visual. Usuários avançados ainda podem burlar desativando JavaScript ou acessando arquivos diretamente. Para impedir de verdade, configure também na hospedagem/servidor, por exemplo Firebase Hosting, Cloudflare, Nginx, Apache ou painel da hospedagem.

3) PREVIEW COM MANUTENÇÃO LIGADA
Com enabled: true, você ainda pode visualizar uma página usando:
?vlMaintenancePreview=1

Exemplo:
index.html?vlMaintenancePreview=1


ATUALIZAÇÃO LEVE DO LOADER
-------------------------
Para evitar travamento e consumo alto, o loader agora usa uma imagem aleatória por abertura e não troca de imagem durante o carregamento.

No arquivo assets/js/velarion.config.js:
- backgroundCount: quantidade total de imagens na pasta.
- rotateDuringLoading: false = mais leve; true = troca imagem durante carregamento.
- backgroundIntervalMs: use 0 no modo leve; se ativar rotação, use 3500 ou mais.
- preloadNextBackground: false = mais leve; true = pré-carrega a próxima imagem.
- maxDurationMs: tempo máximo de segurança para o loader fechar sozinho.

Recomendado para PC antigo/navegador pesado:
rotateDuringLoading: false
backgroundIntervalMs: 0
preloadNextBackground: false
maxDurationMs: 5200
